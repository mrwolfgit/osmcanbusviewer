# OSM CAN Bus Viewer

A single-file, web-based CAN log viewer.  
Upload a `.txt`, `.log`, or `.csv` file containing CAN bus messages and view:

- CAN IDs and message counts
- Message frequency over time (graph)
- Recent frames in a scrollable table
